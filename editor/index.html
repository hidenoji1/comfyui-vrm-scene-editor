モデルとカメラ

ファイル ▾ → モデルを追加
VRMをサムネイルから選んで追加（複数可）。矢印モードでビュー上のモデルをクリックして編集対象を切替。
配置モデル（モデル / ボーン 窓）
表示／非表示・選択・削除（確認あり）。
カメラ設定
カメラ1〜9。表示モード（image / mask / mask(hands) / seg / depth / normal / openpose…）を複数チェックで選択、有効／無効、含めるモデルを指定。
📷 撮影
有効カメラ × 選択した表示モードを一括で保存・転送（camera{n}_{type}）。
seg（セグメンテーション）
体と手を色分け。モデルごとに色相が変わる（含めるモデル数 × 2色）。自前のマスク／合成向け。

ハンドフォーカス（手の詳細編集）

「手のポーズ」窓の「左手を編集 / 右手を編集」で開始。

表示
手首から先だけを拡大表示し、手首を中心に回転。編集中は親IK（腕・体）を固定して手が動かない。
ハンド編集スライダー
握り・カッピング（手のひらの丸み）、各指の曲げ・開き。3Dハンドルでの微調整も併用可。「平手」で全スライダーを戻す。
ハンドポーズパレット
正方形サムネのグリッド。ダブルクリックで適用（左右共通＝自動でミラー）。「追加」で名前を付けて登録、「削除」で確認の上削除。
終了 (Esc)
クリッピング解除・カメラ・表示を元に戻す。

ポーズライブラリ

サムネイル
青いマネキンに各ポーズを当てた正方形サムネ（自動生成・保存）。
ダブルクリック
選択中モデルにポーズを適用。
削除
選択して「削除」（確認あり）。models/pose のファイルを削除。

背景画像

背景画像（ウィンドウ ▾）
参考画像を背景に敷いてポーズ合わせ。不透明度・拡大・回転・位置、モデルより前面表示も可。撮影には写り込みません。

ウィンドウ

各ウィンドウはタイトルでドラッグ移動、右下でサイズ変更できます。ハンド編集・ハンドポーズパレットは位置・サイズを記憶します。
